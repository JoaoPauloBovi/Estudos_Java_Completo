Estudos Java Completo

Este repositório centraliza o desenvolvimento técnico e exercícios práticos realizados durante o curso de formação em Java. O objetivo é consolidar os fundamentos da linguagem e os pilares da Programação Orientada a Objetos.

Objetivos Técnicos

    Implementação de lógica de programação e algoritmos.
    Aplicação de conceitos de POO: Classes, atributos, métodos e encapsulamento.
    Gerenciamento de memória, tratamento de exceções e manipulação de arquivos.
    Estudo de estruturas de dados e coleções nativas da linguagem.

Estrutura do Repositório

    Exercícios de Fixação: Códigos desenvolvidos para validar conceitos teóricos.
    Projetos de Estudo: Implementações práticas de módulos específicos do curso.

Tecnologias e Ferramentas

    Linguagem: Java.

    Ambiente de Desenvolvimento: IDE IntelliJ e Git para versionamento.
